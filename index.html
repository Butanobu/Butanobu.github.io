Epson Printer

Epson make really good printers(citation needed) but they know very little about computers, websites or technical documents by the look of it. It took me a while to get my Epson printer set up in Ubuntu, and I couldn't find everything in one place, so I've summarised the steps here.

  • Add deb http://download.ebz.epson.net/dsc/op/stable/debian/ lsb3.2 main to sources.list or your own .list file in /etc/apt/sources.list.d/
  • Run sudo apt-get update
  • Grab the ID of the key that is missing for their repo.
  • Run sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys <key ID here>
  • Run sudo apt-get update again.
  • Find your printer at Open Printing and get the name of the driver
  • sudo apt-cache search for elements of the driver name, then sudo apt-get install it.
  • Finally, go through the regular Add Printer dialogue in Ubuntu to find and install your printer in the usual way.
